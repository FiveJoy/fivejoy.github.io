1、如何发布博客

  • hexo n “文章标题” | 【hexo n layout_name “文章标题” layout_name 可以选择post/draft/page/默认为Post】
  • 进入\Hexo\source_posts 编辑该md文件
  • hexo g#表示生成
  • hexo d#部署
  • 或者直接【hexo d -g】

2 、更换theme

  • https://hexo.io/themes/ 找到喜欢的theme
  • 进入\Hexo\theme目录 git bash 并执行git clone https://github.com/klugjo/hexo-theme-clean-blog 等待下载完成
  • 打开\Hexo_config.yml 更改theme: hexo-theme-clean-blog !!!注意themes文件夹中的主体文件夹是什么名字，_config.yml中就改成什么【一定要对应！】
  • hexo clean
  • hexo g -d

3 、hexo-theme-clean-blog的使用说明

  • 英文版 https://github.com/klugjo/hexo-theme-clean-blog

4 、Hexo的几个使用命令

  • ① hexo n draft “草稿名称” 使用[draft布局]将会把文档创建到_drafts中且不会发布出去
  • ② hexo public “草稿名称”将草稿发布到post中
  • hexo new photo “My Gallery” ，Hexo在scaffolds目录中去寻找photo.md的模版文件，并基于它创建标题为My Gallery的文章
