继上一个fivejoy.github.io在create以后就一个月没有使用导致最后完全乱套 & 所有记忆和学习过的东西全都是会忘记。这篇就当做备忘录吧

如何发布博客

  • hexo n “文章标题”
  • 进入\Hexo\source_posts 编辑该md文件
  • hexo g#表示生成
  • hexo d#部署
  • 或者直接【hexo d -g】

更换theme

  • https://hexo.io/themes/ 找到喜欢的theme
  • 进入\Hexo\theme目录 git bash 并执行git clone https://github.com/klugjo/hexo-theme-clean-blog 等待下载完成
  • 打开\Hexo_config.yml 更改theme: hexo-theme-clean-blog !!!注意themes文件夹中的主体文件夹是什么名字，_config.yml中就改成什么【一定要对应！】
  • hexo clean
  • hexo g -d

hexo-theme-clean-blog的使用说明

  • 英文版 https://github.com/klugjo/hexo-theme-clean-blog

所以无论是 做了什么修改，最后只要执行hexo g -d即可使修改生效了
